# VidyaJobsAI-# 🚀 VidyaJobsAI V10 Ultra

**VidyaJobsAI** ek advanced AI-powered job search aur career management portal hai. Ye platform users ko latest jobs dhundhne, AI se career guidance lene, aur professional resumes banane mein madad karta hai.

---

## ✨ Key Features

* **🤖 VidyaJobsAI Chatbot:** OpenRouter API (GPT-4o/Gemini) ka use karke career aur jobs se jude sawalon ke jawab.
* **🔍 Smart Job Search:** Serper API ke zariye latest Govt aur Private jobs ki real-time updates.
* **📄 AI Resume Builder:** Sirf details enter karke professional PDF resume generate karein.
* **🖼️ OCR & PDF Reader:** Images se text nikalne (OCR) aur PDF files ko read karne ke liye tools.
* **📱 PWA Ready:** Mobile par app ki tarah install karein (Offline support ke saath).
* **🌓 Dark Mode:** Eye-friendly UI dark mode support ke saath.
* **💬 Community Help:** Live community chat jahan users ek dusre ki help kar sakte hain.

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, JavaScript
- **AI Models:** GPT-4o-mini, Gemini, Mistral (via OpenRouter)
- **APIs:** Serper (Google Search), OpenRouter (LLMs)
- **Deployment:** Gunicorn (Production Server)
